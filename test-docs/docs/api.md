# API Reference

Quick API reference for testing.

## Classes

### `TestClass`

A sample class for testing documentation.

**Methods:**

- `get()` - Retrieve data
- `set(value)` - Store data
- `delete()` - Remove data

**Example:**

```python
obj = TestClass()
obj.set("test value")
result = obj.get()
```

## Functions

### `process_data(input: str) -> dict`

Process input data and return results.

**Parameters:**

- `input` (str): The input string to process

**Returns:**

- `dict`: Processed results

**Example:**

```python
result = process_data("sample")
print(result["status"])
```
